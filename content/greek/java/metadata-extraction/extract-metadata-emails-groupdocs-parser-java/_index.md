### Εξαγωγή Μεταδεδομένων από Αρχεία Email
Αυτή η ενότητα δείχνει πώς να διαβάσετε ένα αρχείο email και να εκτυπώσετε τα μεταδεδομένα του.

#### Βήμα 1: Ρυθμίστε τη Διαδρομή του Αρχείου Σας
Καθορίστε τη θέση του email που θέλετε να επεξεργαστείτε:

```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.msg";
```
Αντικαταστήστε το placeholder με την πραγματική διαδρομή του αρχείου `.msg` σας.

#### Βήμα 2: Αρχικοποιήστε τον Parser και Εξάγετε Μεταδεδομένα
Δημιουργήστε ένα αντικείμενο `Parser`, ανακτήστε τα μεταδεδομένα και εκτυπώστε κάθε στοιχείο:

```java
try (Parser parser = new Parser(filePath)) {
    Iterable<MetadataItem> metadata = parser.getMetadata();
    
    for (MetadataItem item : metadata) {
        System.out.println(String.format("%s: %s", item.getName(), item.getValue()));
    }
} catch (Exception e) {
    System.err.println("Error occurred while extracting metadata: " + e.getMessage());
}
```

- **Παράμετροι** – Η διαδρομή του αρχείου περνάει στον κατασκευαστή `Parser`.  
- **Τιμές Επιστροφής** – Ένα `Iterable<MetadataItem>` που περιέχει ζεύγη όνομα/τιμή.  
- **Σκοπός** – Διαβάζει το email, εξάγει πεδία όπως **From**, **Subject**, **Date**, και τα εκτυπώνει.

#### Συμβουλές Επίλυσης Προβλημάτων
- Επαληθεύστε ότι η μορφή του email υποστηρίζεται (`.msg` ή `.eml`).  
- Βεβαιωθείτε ότι η έκδοση της βιβλιοθήκης στο `pom.xml` ταιριάζει με αυτή που κατεβάσατε.  
- Ελέγξτε ότι όλες οι απαιτούμενες δηλώσεις import είναι παρούσες.
