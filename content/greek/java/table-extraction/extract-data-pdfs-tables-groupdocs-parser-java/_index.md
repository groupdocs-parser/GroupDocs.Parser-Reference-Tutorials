Συμπεριλάβετε τη βιβλιοθήκη στο έργο σας και δημιουργήστε μια παρουσία `Parser`:

```java

public class Main {

            System.out.println("GroupDocs.Parser initialized successfully.");
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

## Οδηγός Βήμα‑βήμα για την Εξαγωγή Δεδομένων από Πίνακα

### Βήμα 1: Ορισμός Παραμέτρων Προτύπου
Δημιουργήστε ένα αντικείμενο `TemplateTableParameters` που περιγράφει τη θέση και το μέγεθος του πίνακα στη σελίδα:

```java
import com.groupdocs.parser.templates.Rectangle;
import com.groupdocs.parser.templates.Size;
import com.groupdocs.parser.templates.Point;

// Specify the path to your document directory
String documentPath = "YOUR_DOCUMENT_DIRECTORY/YourDocument.pdf";

TemplateTableParameters parameters = new TemplateTableParameters(
    new Rectangle(new Point(35, 320), new Size(530, 55)), null);
```

### Βήμα 2: Δημιουργία Προτύπου Πίνακα
Χρησιμοποιήστε τις παραμέτρους για να δημιουργήσετε ένα `TemplateTable`. Το προαιρετικό όνομα βοηθά στην ταυτοποίηση του πίνακα αργότερα:

```java
import com.groupdocs.parser.templates.TemplateTable;

// Define the table with specified parameters
templateTable = new TemplateTable(parameters, "Details");
```

#### Ανάλυση Παραμέτρων
- **Rectangle(Point(35, 320), Size(530, 55))** – επάνω‑αριστερή γωνία (X = 35, Y = 320) και πλάτος/ύψος του πίνακα.  
- **"Details"** – ένα φιλικό αναγνωριστικό που μπορείτε να αναφέρετε κατά την εξαγωγή δεδομένων.

### Βήμα 3: Εξαγωγή Περιεχομένου Πίνακα
Αφού ορίσετε το πρότυπο, μπορείτε να καλέσετε τις μεθόδους εξαγωγής του parser (ο κώδικας παραλείπεται για να διατηρηθεί ο αρχικός αριθμός μπλοκ). Ο parser επιστρέφει γραμμές και κελιά που μπορείτε να αντιστοιχίσετε σε αντικείμενα Java ή να εξάγετε σε CSV/JSON.
