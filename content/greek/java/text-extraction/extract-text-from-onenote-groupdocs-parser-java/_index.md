## Οδηγός Υλοποίησης

### Βήμα 1: Καθορίστε τη Διαδρομή του Εγγράφου
Ορίστε την απόλυτη ή σχετική διαδρομή προς το αρχείο OneNote σας.

```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.one";
```

### Βήμα 2: Αρχικοποιήστε τον Parser
Δημιουργήστε ένα στιγμιότυπο `Parser` μέσα σε μπλοκ try‑with‑resources ώστε να κλείνει αυτόματα.

```java
try (Parser parser = new Parser(filePath)) {
    // Proceed with text extraction using the parser instance.
}
```

*Γιατί είναι σημαντικό*: Η σωστή διαχείριση πόρων αποτρέπει διαρροές μνήμης, ειδικά όταν επεξεργάζεστε μεγάλα σημειωματάρια.

### Βήμα 3: Εξάγετε το Περιεχόμενο Κειμένου
Χρησιμοποιήστε τη μέθοδο `getText()` για να αποκτήσετε ένα `TextReader`. Στη συνέχεια διαβάστε ολόκληρο το περιεχόμενο.

```java
try (TextReader reader = parser.getText()) {
    String extractedText = reader.readToEnd();
    
    // Process or save the text as needed.
}
```

*Γιατί είναι σημαντικό*: Η `getText()` μεταδίδει το κείμενο του σημειωματάριου σε ροή, παρέχοντάς σας ένα ενιαίο string που μπορείτε να αποθηκεύσετε, ευρετηριάσετε ή αναλύσετε.
