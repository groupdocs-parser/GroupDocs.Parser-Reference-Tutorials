## Πώς να εξάγετε κείμενο από Excel χρησιμοποιώντας java parse excel file

### Βήμα 1: Ορίστε τη διαδρομή του αρχείου Excel
Ενημερώστε τον parser πού βρίσκεται το βιβλίο εργασίας σας.

```java
String excelFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.xlsx";
```

### Βήμα 2: Αρχικοποιήστε τον Parser
Δημιουργήστε ένα αντικείμενο `Parser` μέσα σε ένα μπλοκ try‑with‑resources ώστε το χειριστήριο αρχείου να κλείνει αυτόματα.

```java
try (Parser parser = new Parser(excelFilePath)) {
    // Continue to the next step
}
```

### Βήμα 3: Διαβάστε όλο το κείμενο
Καλέστε το `getText()` για να λάβετε ένα `TextReader`, στη συνέχεια εξάγετε όλο το κείμενο του φύλλου σε μια συμβολοσειρά.

```java
try (TextReader reader = parser.getText()) {
    String extractedText = reader.readToEnd();
    System.out.println(extractedText);
}
```

#### Εξήγηση βασικών στοιχείων
- **Parser** – Κύρια κλάση που ανοίγει και ερμηνεύει το βιβλίο εργασίας.  
- **getText()** – Επιστρέφει ένα `TextReader` που ρέει όλες τις τιμές κελιών ως απλό κείμενο.  
- **readToEnd()** – Συλλέγει τα ροή δεδομένων σε ένα ενιαίο `String`.
