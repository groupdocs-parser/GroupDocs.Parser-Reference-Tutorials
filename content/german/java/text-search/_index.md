## Häufig gestellte Fragen

**Q: Kann ich GroupDocs.Parser für Java in einem kommerziellen Projekt verwenden?**  
A: Ja, sobald Sie eine kommerzielle Lizenz erhalten haben, können Sie die Bibliothek in jede Produktionsanwendung integrieren.

**Q: Unterstützt die Bibliothek passwortgeschützte Excel‑Dateien?**  
A: Absolut. Sie können das Passwort beim Öffnen der Arbeitsmappe übergeben, und der Parser entschlüsselt sie on‑the‑fly.

**Q: Wie skaliert die Leistung der Schlüsselwortsuche bei großen Tabellen?**  
A: GroupDocs.Parser liest Daten in Streaming‑Form, sodass der Speicherverbrauch selbst bei Dateien mit tausenden Zeilen gering bleibt. Bei sehr großen Dateien sollten Sie die Verarbeitung Blatt‑für‑Blatt in Betracht ziehen.

**Q: Kann ich Schlüsselwortsuche mit Regex in derselben Abfrage kombinieren?**  
A: Ja. Sie können zunächst einen einfachen Schlüsselwortfilter ausführen und anschließend ein Regex‑Muster auf die passenden Zellen anwenden, um feinere Kontrolle zu erhalten.
