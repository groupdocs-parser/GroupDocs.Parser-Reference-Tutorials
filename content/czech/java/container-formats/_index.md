## Jak procházet ZIP soubory pomocí GroupDocs.Parser
Při práci s velkými nebo vnořenými archivy je efektivní procházení ZIP souborů nezbytné pro výkon a spolehlivost. GroupDocs.Parser nabízí streamingové API, které vám umožní číst každou položku, aniž byste načítali celý archiv do paměti. Tento přístup je ideální pro dávkové zpracování, indexaci obsahu nebo extrakci za běhu.
