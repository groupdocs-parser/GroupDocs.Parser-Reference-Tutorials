## 如何使用 GroupDocs.Parser 遍历 ZIP 文件
在处理大型或嵌套的归档文件时，高效遍历 ZIP 文件对于性能和可靠性至关重要。GroupDocs.Parser 提供了流式 API，允许您在不将整个归档加载到内存中的情况下读取每个条目。这种方法非常适合批处理、内容索引或即时提取场景。
