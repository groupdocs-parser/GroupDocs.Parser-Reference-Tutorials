## Frequently Asked Questions

**Q: क्या मैं एन्क्रिप्टेड या पासवर्ड‑प्रोटेक्टेड PDFs से टेक्स्ट निकाल सकता हूँ?**  
A: हाँ—सिर्फ `Parser` कंस्ट्रक्टर या `load` मेथड में पासवर्ड पास करें, और एक्सट्रैक्शन सामान्य रूप से काम करेगा।

**Q: GroupDocs.Parser कौन‑से आउटपुट फ़ॉर्मेट्स को कन्वर्ज़न के लिए सपोर्ट करता है?**  
A: प्लेन टेक्स्ट, HTML, Markdown, और आप कस्टम फ़ॉर्मेटिंग के लिए लेआउट‑अवेयर टेक्स्ट एरिया भी प्राप्त कर सकते हैं।

**Q: क्या PDF के केवल विशिष्ट पेजों से टेक्स्ट निकालना संभव है?**  
A: बिल्कुल। एक्सट्रैक्शन मेथड को कॉल करने से पहले `PageOptions` क्लास का उपयोग करके पेज रेंज निर्दिष्ट करें।

**Q: “extract PDF text Java” Apache PDFBox से कैसे अलग है?**  
A: GroupDocs.Parser उच्च‑स्तरीय API प्रदान करता है, कई फ़ाइल टाइप्स के लिए बिल्ट‑इन सपोर्ट और जटिल लेआउट्स को संभालने में बेहतर है, जबकि PDFBox जैसी लो‑लेवल लाइब्रेरी कम सुविधाएँ देती हैं।

**Q: मुझे कौन‑सा GroupDocs.Parser संस्करण उपयोग करना चाहिए?**  
A: हमेशा नवीनतम Maven रिलीज़ उपयोग करें; इसमें बग फ़िक्स, परफ़ॉर्मेंस सुधार और नए डॉक्यूमेंट फ़ॉर्मेट्स का समर्थन शामिल होता है।
