## O que é Java Keyword Search Excel?
Java keyword search excel refere‑se ao processo de varrer programaticamente planilhas Excel em busca de palavras ou frases específicas usando código Java. Com o GroupDocs.Parser, você pode ler valores de células sem abrir o Excel, executar correspondências rápidas de palavras‑chave e recuperar a localização exata (planilha, linha, coluna) de cada ocorrência.

## Por que usar o GroupDocs.Parser para busca de texto?
- **Velocidade e eficiência de memória:** Analisa documentos sem carregar o aplicativo Office completo.  
- **API unificada:** O mesmo código funciona para Excel, PDF, Word, PowerPoint, HTML, EPUB e formatos de email.  
- **Recursos avançados:** Suporte a regex, busca página a página e realce de resultados.  
- **Multiplataforma:** Funciona em qualquer ambiente compatível com Java, desde aplicativos desktop até serviços em nuvem.

## Perguntas Frequentes

**Q: Posso usar o GroupDocs.Parser para Java em um projeto comercial?**  
A: Sim, após obter uma licença comercial você pode integrar a biblioteca em qualquer aplicação de produção.

**Q: A biblioteca suporta arquivos Excel protegidos por senha?**  
A: Absolutamente. Você pode passar a senha ao abrir a pasta de trabalho, e o parser a descriptografará em tempo real.

**Q: Como o desempenho da busca por palavras‑chave escala com planilhas grandes?**  
A: O GroupDocs.Parser lê os dados de forma streaming, portanto o uso de memória permanece baixo mesmo para arquivos com milhares de linhas. Para arquivos muito grandes, considere processar planilha por planilha.

**Q: Posso combinar busca por palavras‑chave com regex na mesma consulta?**  
A: Sim. Você pode primeiro executar um filtro simples de palavras‑chave e depois aplicar um padrão regex às células correspondidas para controle mais refinado.
