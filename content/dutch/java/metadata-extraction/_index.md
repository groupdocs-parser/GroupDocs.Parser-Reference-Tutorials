GroupDocs.Parser biedt een eendrachtige API die werkt met PDF‑bestanden, Office‑bestanden, e‑mails en meer. Door de onderstaande tutorials te volgen, leer je hoe je:

* **Documenteigenschappen lezen Java** – haal standaardvelden op zoals titel, onderwerp en trefwoorden.  
* **Aanmaakdatum verkrijgen Java** – verkrijg de tijdstempel wanneer een bestand oorspronkelijk is aangemaakt.  
* **Auteursnaam verkrijgen Java** – ontdek wie het document heeft geschreven, zelfs wanneer de eigenschap verborgen is in aangepaste metadata.

Deze mogelijkheden laten je zoek‑indexen verrijken, nalevingsbeleid afdwingen, of simpelweg bestandsdetails in je UI weergeven.
